### 🛡️ System Administration & DevOps
I specialize in deploying and maintaining **Production-Grade Servers** for webhosting clients. My focus is on the "Three Pillars": **Security, Stability, and Scalability.**

* **Server Hardening:** Implementing Fail2Ban, IPTables/UFW, and SSH key-only authentication to mitigate brute-force attacks.
* **Web Stack Optimization:** Expertise in **LEMP/LAMP** stacks, fine-tuning Nginx and PHP-FPM for high-traffic concurrency.
* **Automation:** Writing Bash scripts to automate backups, monitoring, and repetitive sysadmin tasks.
* **Zero-Downtime Migration:** Seamlessly moving large-scale web applications between data centers with minimal service interruption.
